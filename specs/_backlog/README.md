# Feature Backlog

This directory is the **pre-formalized pipeline** for features. Ideas land here first as lightweight notes, get triaged, prioritized, and — when ready — promoted to a formal `specs/00X-feature/` directory via Spec Kit's `/specify` command.

---

## 📋 How It Works
↓
specs/_backlog/F00X-name.md       ← capture (this directory)
↓
triage weekly, prioritize
↓
specs/00X-feature-name/spec.md    ← promote via /specify
↓
/clarify → /plan → /tasks → /implement
↓
merged, marked "shipped" in backlog item

---
